# Sales Performance Dashboard

## Project Overview

An interactive Power BI dashboard created using the Sample Superstore dataset.

## Key Features

- KPI Cards for Sales, Profit, and Orders
- Region-wise Sales Analysis
- Category-wise Sales Analysis
- Interactive Filters (Region, Category, State)
- Business Performance Insights

## Tools Used

- Power BI
- Excel/CSV Dataset
